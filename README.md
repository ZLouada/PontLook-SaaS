# Pontlook

Premium marketing + lead-generation website for Pontlook, a corporate training matchmaking platform for the GCC.

Two-sided marketplace: corporate training providers ↔ GCC companies with verified workforce challenges.

## Stack

- Next.js (App Router) + TypeScript, `src` directory
- Tailwind CSS (custom design tokens) + glassmorphism utilities
- Framer Motion, Lucide React
- React Hook Form + Zod (multi-step wizard with localStorage persistence)
- Local MDX blog (Insights)

## Development

```bash
npm install
npm run dev
```

## Design tokens

- Primary `#2451BF`, interactive `#3D7BFF`, accent `#7FB8FF`
- Backgrounds: white with `#F4F7FF` gradients
- Headings: Poppins · Body: Inter

> Market statistics used on the site are illustrative placeholders, flagged for easy replacement once verified figures are available.
